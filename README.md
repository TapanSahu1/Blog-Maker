# Blog-Maker
A personal Blog writing website 

To use the website download the files in your computer and run "npm i" on the command line to install all the node modules.

Remember to pre instal Node.js in your system to run  this.

After installing the necessary modules through "npm i", run "node index.js" or "nodemon index.js" to run thhe program.

To access the website go to any browser and type "localhost:3000". 

You can stop the server by typing "ctrl+c" on the same directory to stop the server.
